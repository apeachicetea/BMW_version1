
# BMW X Tesla

평소에 차량에 관심이 많은데, 그 중에 BMW, Tesla에 더욱 관심이 많다.<br/>그동안 배운 지식으로 좋아하는 것들을 활용해볼 수 없을까 생각해보니 차량 관련 홈페이지를 만들어보고 싶어졌다.<br/>
로고 및 차량 사진들은 BMW, 홈페이지 구조는 Tesla를 참고했다.
